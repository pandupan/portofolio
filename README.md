# 💼 Portofolio

Website portofolio pribadi untuk menampilkan proyek, keahlian, dan pengalaman dalam dunia pengembangan web dan teknologi.

## ✨ Fitur

- **Profil** — Informasi pribadi dan latar belakang
- **Proyek** — Galeri proyek yang pernah dikerjakan
- **Keahlian** — Tech stack dan kemampuan teknis
- **Kontak** — Formulir kontak

## 🛠️ Tech Stack

- **Next.js** — React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
